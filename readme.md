⚡ Overview

This directory houses the innux visual core, designed for live-streamed terminal projections at Hayfilm Studio. It transforms raw system output into rhythmic, industrial-grade ASCII art synchronized with the frequency of underground techno.

🏗 Contents

visuals.py: The main animation engine (Python).

config.json: Aesthetic parameters (colors, characters, speed).

README.md: This document.

🚀 Deployment

To initiate the visual stream on the projector:

Configure Terminal:

Background: #000000 (Black)

Font: Monospace Bold (16pt or higher)

Fullscreen: Press F11

Launch Engine:

python3 visuals.py


🕹 Live Controls

Ctrl + C: Emergency Stop (Exit to Shell).

Shift + R: Reload configuration (if implemented).
