# MOCT | HAYFILM CLUSTER
### Terminal Visual Engine v1.0

## ⚡ Overview
This directory houses the **innux** visual core for live projections at **Hayfilm Studio**. It transforms the terminal into a VJ instrument for industrial techno environments.

## 🧰 The VJ Toolkit (Top 3 Tools)
1. **Python (`visuals.py`):** Smooth, math-based pulses and "Lava Lamp" effects.
2. **Bash (`matrix.sh`):** High-speed industrial "Code-Rain" and system noise.
3. **Node.js (`glitch.js`):** Rapid, stuttering ASCII particle glitches.

## 🚀 Deployment
1. **Terminal:** Black background, Monospace Bold font (16pt+), F11 for Fullscreen.
2. **Launch:**
   python3 innux/visuals.py
   bash innux/matrix.sh
   node innux/glitch.js
